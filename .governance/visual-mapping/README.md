## PURPOSE

This folder holds visual representations of the POE architecture —
HTML diagrams, system maps, and visual reference docs that aid
cognitive orientation during build sessions.

These are NOT governance specs. They are visual companions to the
governance system. They do not govern build decisions — they explain them.

## CONTENTS

### poe-architecture.html
Full system architecture map showing:
- LIVE stack (S-001 through S-009, all deployed components)
- Plumbing connections (input → AI → tensor engine → renderer → persistence)
- Open gate: S-008/S-009 output test
- NEXT: SLICE-VIZ-DATASPHERE spec + 4 projection lens modes
- INCUBATING: SLICE-SOCRATIC → META-LAYER dependency chain
- FUTURE: Longitudinal tracking, Auth, Workshop Mode
- Status legend: Live / Gate / Next / Incubating / Future

Open in any browser. No server required.
